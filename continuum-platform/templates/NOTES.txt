Continuum Platform has been deployed.

Components:
  - Cloud Gateway (reverse proxy / single entry point)
  - Cluster Manager (workbench lifecycle)
  - API Server
  - Orchestration Service
  - Message Bridge
  - Feature Base Worker
  - Feature Cheminformatics Worker

To verify the deployment:

  kubectl get pods -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

Access the platform:

  # Cloud Gateway (single entry point)
  kubectl port-forward svc/{{ include "continuum-platform.cloud-gateway.fullname" . }} 8080:8080 -n {{ .Release.Namespace }}

  # Open the UI
  # http://localhost:8080/cluster-manager/ui/
